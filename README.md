<h1 align="center">📚 Aplikasi Peminjaman Buku</h1>

<p align="center">
Aplikasi peminjaman buku berbasis Java Desktop menggunakan Apache NetBeans dan MySQL.
</p>

<hr>

<h2>✨ Fitur</h2>

<ul>
  <li>Login Admin</li>
  <li>Manajemen Data Buku</li>
  <li>Manajemen Data Anggota</li>
  <li>Transaksi Peminjaman Buku</li>
  <li>Transaksi Pengembalian Buku</li>
  <li>Pencarian Data</li>
  <li>Database MySQL</li>
</ul>

<hr>

<h2>🛠️ Persyaratan Sistem</h2>

<p>Sebelum menjalankan project ini, pastikan software berikut sudah terinstall:</p>

<ul>
  <li>XAMPP / MySQL Server</li>
  <li>JDK 7, JDK 8, atau versi terbaru</li>
  <li>Apache NetBeans</li>
</ul>

<hr>

<h2>🗄️ Cara Import Database</h2>

<ol>
  <li>Jalankan <b>XAMPP</b> lalu aktifkan:
    <ul>
      <li>Apache</li>
      <li>MySQL</li>
    </ul>
  </li>

  <li>Buka browser dan akses:
    <pre>http://localhost/phpmyadmin</pre>
  </li>

  <li>Buat database baru dengan nama:
    <pre>perpustakaan</pre>
  </li>

  <li>Pilih database <b>perpustakaan</b></li>

  <li>Klik tab <b>Import</b></li>

  <li>Klik <b>Choose File</b> lalu pilih file:
    <pre>perpustakaan.sql</pre>
  </li>

  <li>Klik tombol <b>Go</b></li>

  <li>Database akan terimport secara otomatis</li>
</ol>

<hr>

<h2>▶️ Cara Menjalankan Aplikasi</h2>

<h3>Menggunakan Apache NetBeans</h3>

<ol>
  <li>Buka <b>Apache NetBeans</b></li>
  <li>Klik <b>Open Project</b></li>
  <li>Pilih folder project aplikasi</li>
  <li>Pastikan <b>XAMPP/MySQL</b> sudah aktif</li>
  <li>Klik tombol <b>Run Project</b></li>
  <li>Aplikasi siap digunakan</li>
</ol>

<h3>Menggunakan File Installer</h3>

<ol>
  <li>Jalankan file:
    <pre>perpustakaan-x86_64.exe</pre>
  </li>

  <li>Ikuti proses instalasi</li>
  <li>Setelah selesai, aplikasi siap digunakan</li>
</ol>

<h2>📌 Catatan</h2>

<p>Jika terjadi error koneksi database:</p>

<ul>
  <li>Pastikan MySQL pada XAMPP sudah berjalan</li>
  <li>Pastikan nama database adalah:
    <pre>perpustakaan</pre>
  </li>
  <li>Pastikan file <code>.sql</code> sudah berhasil diimport</li>
</ul>
